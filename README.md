# 🛒 Exploratory Data Analysis on Retail Sales Data

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on retail sales data to uncover patterns, trends, and insights that can help improve business decisions.

## 🎯 Objectives
- Clean and preprocess the dataset  
- Analyze sales across categories and regions  
- Identify top-performing products  
- Visualize trends and patterns  

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

## 📊 Key Insights
- Sales vary across different regions and categories  
- Some products generate high sales but low profit  
- Seasonal trends impact overall performance  

## 📁 Files Included
- `retail_analysis.ipynb` → Main project file  
- `README.md` → Project description  

## 🚀 Conclusion
EDA helps in understanding customer behavior and improving sales strategies through data-driven insights.
